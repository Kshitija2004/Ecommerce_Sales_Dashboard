# Ecommerce_Sales_Dashboard

This project is an interactive Ecommerce Sales Dashboard created using Power BI to analyze sales performance of Madhav Ecommerce.
The dashboard provides insights into revenue, profit, quantity sold, customer behavior, payment modes, and seasonal trends, enabling data-driven business decisions.

Objectives:

Analyze overall sales and profitability.

Track key performance indicators. (KPIs)

Identify top-performing states, customers, and product categories.

Understand monthly profit trends.

Analyze customer payment mode preferences.
